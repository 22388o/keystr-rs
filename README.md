# **Keystr**: Nostr Keystore

An application for managing Nostr keys.
Written in Rust, with simple UI (Iced).

## Roadmap

- Safekeeping of keys
- Delegation
- Profile
- Signer

## Building and Running

- Prerequisite: `rust`, >= 1.64

- Simply run:  `cargo run`
